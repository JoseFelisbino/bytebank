# 💰 Bytebank

Sistema simples de controle de transações financeiras desenvolvido com **TypeScript**, permitindo registrar depósitos, transferências e pagamentos, atualizando o saldo em tempo real.

---

## 🚀 Acesse o projeto

👉 [Clique aqui para acessar o projeto](https://bytebank-five.vercel.app/)

---

## 📌 Funcionalidades

- ✅ Cadastro de transações financeiras
- ✅ Tipos de transação:
  - Depósito
  - Transferência
  - Pagamento de boleto
- ✅ Atualização automática do saldo
- ✅ Validação de formulário
- ✅ Manipulação do DOM com JavaScript/TypeScript
- ✅ Organização modular do código

---

## 🛠️ Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **TypeScript**

```
bytebank/
```

## ⚙️ Como executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/JoseFelisbino/bytebank.git
```

### 2. Acessar a pasta do projeto

```bash
cd bytebank
```

### 3. Compilar o TypeScript

```bash
tsc
```

### 4. Executar o projeto

```bash
dist/index.html
```
